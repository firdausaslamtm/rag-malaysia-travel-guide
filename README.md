# RAG Malaysia Travel Guide

Retrieval-Augmented Generation chatbot that answers questions about Malaysia travel using public tourism PDFs and websites.

Tech: Python, OpenAI/Azure OpenAI, LangChain, Chroma, FastAPI

Features:
- Ingest public domain travel guides
- Chunk, embed, store in vector DB
- Retrieve relevant passages and generate answers with citations
- Simple web UI

Data sources: Tourism Malaysia public brochures, Wikipedia extracts, OpenStreetMap descriptions.

No company data used.
